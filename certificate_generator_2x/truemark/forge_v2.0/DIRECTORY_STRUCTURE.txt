═══════════════════════════════════════════════════════════════════════════
                          QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════

STEP 1: Installation (5 minutes)
    cd "T:\certificate generator 2x\truemark\forge_v2.0"
    .\install.ps1

STEP 2: Verify Installation
    python certificate_forge.py stats

STEP 3: Mint First Certificate
    python certificate_forge.py mint `
      --owner "Your Name" `
      --wallet "0xYourWalletAddress" `
      --title "Your Asset Title" `
      --ipfs "ipfs://YourIPFSHash" `
      --category "Knowledge"

STEP 4: Verify Certificate
    python certificate_forge.py verify --serial "DALS..."

STEP 5: View Output
    Open: T:\certificate generator 2x\Vault_System_1.0\certificates\issued\
