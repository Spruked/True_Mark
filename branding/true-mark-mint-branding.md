## 🟦 Brand Description (Short)

```
True Mark Mint is a forensic-grade minting engine for knowledge, legacy, and enterprise assets. It provides secure issuance, off-chain storage, optional encryption, and printable certificates backed by Polygon or Ethereum.
```

---

## 🟦 Brand Description (Long)
```
True Mark Mint is a full-stack, forensic-grade minting engine designed for knowledge assets, legacy preservation, and enterprise certification. It supports K-NFT, H-NFT, L-NFT, and C-NFT types with optional encryption, off-chain storage, dynamic pricing, and printable certificates. Users can mint with fiat or crypto without needing a wallet, while enterprises can deploy to Polygon or Ethereum with full governance, royalties, and inheritance logic.
```
